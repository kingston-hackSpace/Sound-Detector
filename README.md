# Sound Detector

DESCRIPTION
- 
The SparkFun Sound Detector is a microphone-based sensor that detects the presence and level of sound
in the environment.

*It’s NOT a sound recorder—* it only measures sound intensity (volume) and provides a corresponding signal.
